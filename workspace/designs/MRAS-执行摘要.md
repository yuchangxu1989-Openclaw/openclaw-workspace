## 核心设计亮点

### 1. 架构创新
- **双层架构**：主Agent直通 + 子Agent智能路由
- **非侵入式**：现有代码无需修改，子Agent按需接入

### 2. 与另一AI方案对比
| 维度 | 关键词方案 | MRAS改进 |
|:---|:---|:---|
| 识别方式 | 关键词匹配 | 语义意图识别 |
| 准确率 | 中 | 高 |
| 个性化 | 不支持 | 子Agent独立配置 |
| 韧性降级 | 自建 | **100%复用LEP** |
| 扩展性 | 改代码 | 配置即生效 |

### 3. LEP复用设计
```
MRAS只负责：意图识别 + 模型选择 + 生成降级链建议
LEP负责全部：熔断 + 重试 + 降级执行 + 追踪 + 日志
```

### 4. 配置化设计 (N019/N020模式)
```
Layer 4: 子Agent个性化配置 (最高优先级)
Layer 3: 任务类型默认配置
Layer 2: 全局默认配置  
Layer 1: CapabilityAnchor能力锚点 (只读)
```

---

## 沙盒三层机制

```
Layer 1: 健康检查 (50ms超时，不健康直接跳过)
Layer 2: 影子测试 (1%旁路验证新模型)
Layer 3: 超时熔断 (5s连接/60s响应，LEP原生)
```
