## Day2: 意图触发 → 规则化自动链路
**目标**：用户说"规则化X"→ 系统自动完成感知/认知/执行三层代码交付
**交付物**：
1. **意图识别层**：识别"规则化"类意图，提取规则化目标对象
2. **ruleify执行引擎**：自动生成三层代码骨架（探针+判断+执行）
3. **注册集成器**：自动将生成的代码注册到git hook / cron / 事件总线
4. **验证器**：自动触发一次验证全链条跑通
**验收标准**：对一个新概念说"规则化"→ 系统自动产出可运行的闭环代码并注册，无需手动干预
