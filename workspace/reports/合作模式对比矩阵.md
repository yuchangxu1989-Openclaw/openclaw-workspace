## 详细对比

### 1️⃣ 软件授权模式 (🏆 首选)

```
┌─────────────────────────────────────────────────────────┐
│  模式：SDK/API 授权，厂商集成到设备或 APP                    │
├─────────────────────────────────────────────────────────┤
│  收费：授权费 50-200 万 + 分成 5-10% 或 纯分成 15-25%           │
├─────────────────────────────────────────────────────────┤
│  适合：中小厂商、快速切入、降低门槛                         │
├─────────────────────────────────────────────────────────┤
│  优势：快速变现、风险共担、易于扩展                         │
├─────────────────────────────────────────────────────────┤
│  劣势：技术泄露风险、依赖厂商渠道                          │
├─────────────────────────────────────────────────────────┤
│  目标客户：爱国者、纽曼、OEM 厂商等                         │
└─────────────────────────────────────────────────────────┘
```

**财务模型示例**:
```
签约 10 家厂商，平均授权费 100 万 = 1000 万一次性收入
每台设备分成 20 元，年出货 50 万台 = 1000 万/年持续收入
```

---

### 2️⃣ 云服务订阅模式 (🏆 核心)

```
┌─────────────────────────────────────────────────────────┐
│  模式：设备录音，云端 AI 处理，用户订阅服务                    │
├─────────────────────────────────────────────────────────┤
│  收费：个人 9.9-29.9 元/月 | 专业 49-99 元/月 | 企业定制      │
├─────────────────────────────────────────────────────────┤
│  适合：所有厂商、持续性收入、高毛利                        │
├─────────────────────────────────────────────────────────┤
│  优势：SaaS 模式、高毛利、用户粘性高、可交叉销售               │
├─────────────────────────────────────────────────────────┤
│  劣势：前期投入大、需要用户教育、竞争较激烈                  │
├─────────────────────────────────────────────────────────┤
│  目标客户：所有设备厂商 (作为增值服务)                       │
└─────────────────────────────────────────────────────────┘
```

**财务模型示例**:
```
10 万月活用户，ARPU 30 元/月 = 300 万/月收入 = 3600 万/年
毛利率 90%+，边际成本低
```

---

### 3️⃣ 硬件集成模式 (🥉 战略级)

```
┌─────────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────────┤
│  收费：技术入股分红 | 联合研发收益分成 | 定制硬件销售分成      │
├─────────────────────────────────────────────────────────┤
│  适合：头部厂商、战略伙伴、长期绑定                         │
├─────────────────────────────────────────────────────────┤
│  优势：深度绑定、竞争壁垒高、可能股权收益                    │
├─────────────────────────────────────────────────────────┤
│  劣势：投入大、周期长、风险高                             │
├─────────────────────────────────────────────────────────┤
│  目标客户：Sony、华为、小米、百度等                         │
└─────────────────────────────────────────────────────────┘
```

**财务模型示例**:
```
技术入股新公司 20%，3 年后估值 10 亿 = 2 亿股权价值
或 产品销售分成 10%，年销 100 万台×500 元 = 5000 万/年分成
```

## 谈判要点清单

### 软件授权谈判
- [ ] 授权范围 (独家/非独家、地域限制)
- [ ] 授权期限 (通常 2-3 年)
- [ ] 分成基数 (出厂价/零售价/收入)
- [ ] 最低保底 (可选)
- [ ] 技术支持 SLA
- [ ] 数据归属与使用权限

### 云服务谈判
- [ ] 贴牌权限 (厂商品牌 vs OpenClaw 品牌)
- [ ] 收入分成比例 (通常 30-50% 给厂商)
- [ ] 用户数据归属
- [ ] 定价权 (联合定价 vs 厂商自主)
- [ ] 客户支持责任划分
- [ ] SLA 与赔偿条款

### 硬件集成谈判
- [ ] 知识产权归属 (联合开发 vs 各自所有)
- [ ] 投资比例与股权结构
- [ ] 决策机制 (董事会席位)
- [ ] 退出机制 (回购条款、IPO 安排)
- [ ] 竞业限制
- [ ] 资源投入承诺

## 风险预警信号

🚩 **出现以下情况需重新评估合作**:

| 信号 | 风险 | 应对 |
|-----|------|------|
| 厂商要求独家合作 | 限制扩展能力 | 拒绝或要求高额保底 |
| 厂商自建 AI 团队 | 未来竞争风险 | 缩短合同期限，增加退出条款 |
| 分成比例低于 20% | 收益不足 | 提高授权费或拒绝 |
| 数据归属不明确 | 合规风险 | 明确合同条款，要求合规承诺 |
| 厂商渠道能力弱 | 出货量不达预期 | 设置最低出货量要求 |
| 付款周期过长 | 现金流风险 | 要求预付款或缩短账期 |

## 决策流程图

```
开始
  │
  ▼
厂商类型？
  ├─ 头部厂商 (Sony/华为/小米) → 硬件集成 + 云服务
  ├─ 中型厂商 (爱国者/纽曼)   → 软件授权 + 云服务
  ├─ 小型厂商/OEM            → 软件授权 (低门槛)
  └─ 互联网公司有硬件意向     → 联合品牌 + 云服务
  │
  ▼
谈判优先级？
  ├─ 高 (战略意义) → 灵活条件，快速签约
  ├─ 中 (常规合作) → 标准条款，正常流程
  └─ 低 (试探性)   → 严格条款，控制风险
  │
  ▼
签约执行
  │
  ▼
定期复盘 (季度)
  ├─ 出货量达标？ → 继续/深化合作
  └─ 未达标？     → 重新谈判/终止
```
