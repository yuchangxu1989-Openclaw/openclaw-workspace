# ISC智能标准中心 - 文档质量评估系统

## 功能概述

ISC（智能标准中心）文档质量评估系统，为技能生态提供标准化的文档质量检测与评分服务。

## 安装

无需安装，直接运行：

```bash
node /root/.openclaw/workspace/skills/isc-document-quality/index.js [技能路径] [--report]
```

## 使用方法

### 评估单个技能

```bash
# 评估指定技能
node index.js /path/to/skill

# 评估并生成报告文件
node index.js /path/to/skill --report
```

### 自评估

```bash
# 评估当前技能自身
cd /root/.openclaw/workspace/skills/isc-document-quality
node index.js
```

## 评估维度

系统从四个维度评估文档质量：

1. **基础完整性** (40分) - SKILL.md 和 README.md 的结构完整性
2. **规范符合度** (30分) - 命名、格式、语言规范
3. **内容准确性** (20分) - 描述准确性和时效性
4. **扩展完整性** (10分) - 可选字段和代码注释

## 输出示例

```
============================================================
ISC智能标准中心 - 文档质量评估报告
============================================================

【基本信息】
技能名称: example-skill
评估时间: 2026-02-23T00:00:00.000Z

【评估结果】
总分: 85/100
评级: 良好 (B)
```

## 评分标准

- 90-100分：优秀 (A)
- 80-89分：良好 (B)
- 70-79分：合格 (C)
- <70分：需改进 (D)

## 配置

评估标准在 `index.js` 中的 `ISC_CONFIG` 对象定义，可根据需要调整权重。

## 依赖

- Node.js >= 14.0.0
- 无外部依赖
